TROUBLESHOOTING:
================================================================================

ISSUE: "Python is not recognized as an internal or external command"
SOLUTION:
  1. Make sure Python is installed: https://www.python.org/
  2. During installation, CHECK "Add Python to PATH"
  3. Restart Command Prompt/PowerShell
  4. Try: python --version
  5. If not working, try: python3 --version

ISSUE: Permission Denied Errors
SOLUTION:
  1. Run Command Prompt as Administrator:
     - Right-click cmd → Run as Administrator
  2. Check OneDrive sync status:
     - Wait for files to sync before running again
  3. Try a different folder if issue persists

ISSUE: "Cannot find path" error
SOLUTION:
  1. Verify the path exists: C:\Users\shrus\OneDrive\Desktop\IMS
  2. Check that IMS folder exists
  3. If not, create it manually first using File Explorer
  4. Then run the script

ISSUE: Script runs but directories aren't visible
SOLUTION:
  1. Close and reopen File Explorer
  2. Press F5 to refresh
  3. Check if OneDrive is syncing (look for sync icon)
  4. Wait a few seconds and refresh again

ISSUE: "ModuleNotFoundError" or import errors
SOLUTION:
  1. Make sure you're using Python 3.6 or later
  2. Check: python --version
  3. The script only uses standard library (no external packages needed)

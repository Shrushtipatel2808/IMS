PROBLEM: "Python is not recognized"
SOLUTION:
  1. Install Python: https://www.python.org/downloads/
  2. *** IMPORTANT: Check "Add Python to PATH" during installation ***
  3. Restart Command Prompt after installation
  4. Try again: python --version

PROBLEM: Permission Denied
SOLUTION:
  1. Run Command Prompt as Administrator:
     - Right-click Command Prompt
     - Select "Run as Administrator"
  2. Check OneDrive sync status

PROBLEM: Path Not Found
SOLUTION:
  1. Make sure IMS folder exists in the correct location
  2. If missing, create it manually first
  3. Then run the script

PROBLEM: Directories not visible in File Explorer
SOLUTION:
  1. Press F5 to refresh
  2. Close and reopen File Explorer
  3. Wait for OneDrive to finish syncing

PROBLEM: Script runs but appears to do nothing
SOLUTION:
  1. Wait - scripts can take a few seconds
  2. Check if Command Prompt shows output
  3. Look for creation_output.txt file
  4. Verify in File Explorer

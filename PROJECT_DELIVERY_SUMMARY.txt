TROUBLESHOOTING:
================================================================================

PROBLEM: "Python is not recognized"
SOLUTION:
  1. Install Python: https://www.python.org/downloads/
  2. IMPORTANT: Check "Add Python to PATH" during installation
  3. Restart Command Prompt
  4. Try: python --version

PROBLEM: Permission Denied
SOLUTION:
  1. Run Command Prompt as Administrator
  2. Check OneDrive sync status
  3. Try creating in different location if needed

PROBLEM: Path Not Found
SOLUTION:
  1. Verify IMS folder exists
  2. Create it if missing: C:\Users\shrus\OneDrive\Desktop\IMS
  3. Then run the script

PROBLEM: Directories not visible in File Explorer
SOLUTION:
  1. Press F5 to refresh File Explorer
  2. Wait for OneDrive sync
  3. Close and reopen File Explorer
  4. Check that the script completed successfully

PROBLEM: Script runs but shows warnings
SOLUTION:
  1. Most warnings are not critical
  2. Check if ✓ SUCCESS message appears
  3. Verify directories in File Explorer
  4. If directories exist, creation was successful
